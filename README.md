# Github plugins

Check the `packages` directory for all plugins.

These plugins are also available on
[pub](https://pub.dev/flutter/packages).

## Issues

Please file any issues, bugs, or feature requests in the [main
repo](https://github.com/gitterapp/github-flutter/issues/new).

## Contributing

If you wish to contribute a change to any of the existing plugins in this repo,
please review our [contribution guide](https://github.com/gitterapp/github-flutter/blob/master/CONTRIBUTING.md),
and send a [pull request](https://github.com/gitterapp/github-flutter/pulls).
